# 🎉 Interactive Birthday Website 🎉

A fun, interactive birthday surprise website for your best friend!

## Features ✨

✅ **Countdown Timer** - Displays days, hours, minutes, and seconds until July 12
✅ **Customizable Cake** - Choose flavors, frosting colors, toppings, and candles
✅ **Interactive Gifts** - Three special surprises that reveal one by one
✅ **Personal Letter** - Write and save a heartfelt message
✅ **Photo Gallery** - Add your favorite photos with aesthetic styling
✅ **YouTube Video** - Embed a special video for your friend
✅ **Animations** - Floating balloons, swaying flowers, and falling hearts
✅ **Background Music** - Festive music plays in the background
✅ **Responsive Design** - Works on desktop and mobile devices
✅ **Data Persistence** - All customizations are saved locally

## How to Use 🚀

1. Visit the website
2. Check the countdown timer to the birthday date
3. Customize a beautiful cake with your friend's favorite flavors and toppings
4. Discover three special gifts:
   - 💌 A personalized letter
   - 📸 A photo gallery of memories
   - 🎥 A special YouTube video
5. Enjoy the animations and music!

## Customization 🎨

### Change the Birthday Date
Edit `script.js` line 21:
```javascript
const targetDate = new Date('2026-07-12').getTime();
```
Change `2026-07-12` to your friend's birthday date.

### Change the Background Music
Edit `index.html` line 12:
```html
<source src="https://www.soundhelix.com/examples/mp3/SoundHelix-Song-1.mp3" type="audio/mpeg">
```
Replace with your favorite song URL.

### Edit the Default Letter
Edit `index.html` lines 129-134 to personalize the letter.

## Deployment 🌐

### GitHub Pages
1. Go to repository Settings
2. Scroll to Pages section
3. Select "main" branch as source
4. Your site will be live at: `https://yourusername.github.io/birthday-website`

### Netlify
1. Go to netlify.com
2. Sign up with GitHub
3. Import this repository
4. Your site will be live in seconds!

### Vercel
1. Go to vercel.com
2. Sign up with GitHub
3. Import this repository
4. Your site will be deployed instantly!
